# 多元函数的定义

多元函数是指依赖于多个自变量的函数

- 表示形式：$f(x_1, x_2, ..., x_n)$，其中 $x_1, x_2, ..., x_n$ 是自变量，$f$ 是因变量。
- 定义域与值域：函数的自变量范围和函数值的范围。

在多元函数中，偏导数用于衡量函数对于其中一个自变量的变化率，而其他自变量保持不变。
